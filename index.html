Know Your Phone

Lets know more details of your android phone
download .ZIPdownload .TGZ

Description

Knowing your phone in-detail is a very important aspect for every mobile user. In Android Operating System, there are more facilities to view what is exactly your phone.

We are thinking of providing more details to the user with our mobile application.

We are only using the existing packages provided by Android and displaying each and every possible functionality/feature of your phone.

Yes, we may request lot of phone permissions to get the full list of features of your phone. We can explain why we need that feature and what we are displaying to you by using the feature.

Features

Telephony

Package: android.telephony

Permission: android.permission.READ_PHONE_STATE

Classes:

  • CellLocation
  • TelephonyManager

Content:

  • Phone Type (GSM, CDMA)
  • Device ID (IMEI/MEID)
  • Sofware Version
  • Phone Number
  • SIM Serial Number
  • SIM Subscriber ID
  • SIM Operator Name
  • SIM Operator Code
  • SIM Country ISO Code
  • SIM State
  • Data Activity (IN, OUT, INOUT)
  • Data State (CONNECTED, CONNECTING, DISCONNECTED, SUSPENDED)
